## For runing this project
## install dependencies
```bash
npm install
```
or
```bash
yarn install
```

## Available commands

### start
```bash
yarn start || npm start
```

### build
```bash
yarn build || npm run build
```

### serve
```bash
yarn serve || npm run serve
```
